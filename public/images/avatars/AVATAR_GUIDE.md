# Personality Sim Avatar Guide

This folder stores fixed avatar assets for the final persona report. The app does not call an image API and does not generate images at runtime. Generate or draw the images separately, then replace the matching PNG files in this folder.

## File Names

- `creator.png`
- `observer.png`
- `strategist.png`
- `pleaser.png`
- `avenger.png`
- `avoider.png`
- `caregiver.png`
- `judge.png`
- `default.png`

## Shared Style Requirement

Every avatar prompt must include:

`cute pixel art character, chibi, soft colors, cozy indie game style, expressive face, adorable, non-realistic, clean simple background, 64x64 or 128x128 pixel avatar, unified character proportions, warm neutral palette, gentle outline, screenshot-friendly`

Overall tone:

- Cute and soft
- Like small personality characters from one indie game
- No horror
- No realism
- No hardcore RPG enemy feeling
- Keep all characters visually consistent
